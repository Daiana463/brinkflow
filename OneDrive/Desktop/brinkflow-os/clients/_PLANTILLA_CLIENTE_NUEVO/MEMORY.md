# MEMORY — [NOMBRE DEL CLIENTE]

> **Qué es esto:** la memoria viva del trabajo con este cliente. Lo que aprendimos, lo que funcionó, lo que falló, lo que decidimos.
> **Cuándo se actualiza:** después de cada ciclo de trabajo significativo (campaña lanzada, mes cerrado, decisión importante tomada).
> **Quién lo actualiza:** Vero. Claude puede borronear, Vero edita y aprueba. Esta es la "biblioteca cognitiva" del cliente — sin esto, el sistema reinicia contexto cada vez.
> **Tiempo estimado de actualización:** 10-15 min por ciclo.
